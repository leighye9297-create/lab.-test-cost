# 无缝管实验费用计算系统

> 在线版本：https://你的用户名.github.io/pipe-test-cost-calculator/

## 功能介绍

本系统用于计算无缝管实验费用与材料损耗，支持：

- ✅ 上传物料清单 Excel 自动解析
- ✅ 按生产标准自动匹配实验要求（GB、ASTM、EN）
- ✅ 银隆实验室 / 松阳CNAS 实验室收费标准
- ✅ 组批规则自动计算（按标准+外径+壁厚）
- ✅ 线切割加工费用计算
- ✅ 材料损耗长度优化（共享试样段）
- ✅ 计算结果导出 Excel

## 使用说明

1. 打开网页后，点击 **"📦 加载测试物料清单"** 查看示例
2. 或点击 **"📥 上传物料清单"** 上传自己的 Excel
3. 切换到 **"🧮 费用计算"** 标签页
4. 选择实验室（银隆 / 松阳CNAS）
5. 点击 **"🔬 开始计算"**
6. 查看 **"📋 计算结果"** 并导出 Excel
